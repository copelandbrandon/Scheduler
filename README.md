## Getting Started

1. Install dependencies with `npm install`.

2. Run Webpack Development Server With
```sh
npm start
```
3. Run The Jest Test Framework With
```sh
npm test
```
4. Run Storybook Visual Testbed With
```sh
npm run storybook
```
